Câu lệnh:
call [from] [to]: Tạo cuộc gọi từ số [from] đến số [to].
filter content [content]: Chỉ hiện những log có chứa nội dung [content]. (Nếu filter event tồn tại thì thêm điều kiện event = [event])
filter event [event]: Chỉ hiện những log có event là [event]. (Nếu filter content tồn tại thì thêm điều kiện content = [content])
filter reset: Cài đặt lại tất cả filters.
clear: Dọn dẹp nội dung trên terminal
